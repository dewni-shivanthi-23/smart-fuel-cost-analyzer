# Smart Fuel Cost Analyzer (Sri Lanka)

A machine learning-powered web application that predicts fuel costs based on distance, vehicle type, and fuel price.

## 🚀 Features
- Predict daily and weekly fuel costs
- Supports multiple vehicle types
- Smart cost-saving suggestions
- Clean UI using Streamlit

## 🧠 ML Models Used
- Linear Regression
- Random Forest (selected based on performance)

## 🛠️ Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Docker & Docker Compose

## ▶️ How to Run

```bash
docker-compose up --build
